Визуальная новелла "Закусочная Тонкацу"

Лицензия: MIT

Язык программирования: Python 3.8+

Используемый движок/библиотека: Pygame (или возможна альтернативная)

О проекте Фан-проект по вселенной Bread Barbershop — визуальная новелла, сочетающая хоррор и абсурдную комедию в стилистике аниме и гротескных мультяшных ужасов. Действие происходит в закусочной «Тонкацу», где персонажи, похожие на выпечку и фастфуд, готовят секретное блюдо — острую свиную котлету — и раскрывают ужасные тайны города Бейкери-тауна.

Особенности: Жанр: хоррор + абсурдная комедия

Платформа: ПК (Windows, macOS, Linux)

Движок: Python + Pygame

Длина: 3–4 главы, 2–3 часа игрового времени

Механики: нелинейный сюжет, система страха, мини-игры

Стиль: аниме + гротеск, мультяшный хоррор

Абонентская база: фанаты Bread Barbershop, хоррор-любители, старше 18+

Установка и запуск

Требования

Python: 3.8 и выше

Библиотека: Pygame (поддержка 2.5+, рекомендуется 2.0+)

Дополнительно: комфортный редактор кода (VS Code, PyCharm или иной)

Установка 1.Установите Python с официального сайта: https://python.org 2.Установите Pygame через pip: pip install pygame

3.Скачать проект: git clone https://github.com/ваш-аккаунт/tonkatsu-diner-python.git cd tonkatsu-diner-python

4.Запуск: python main.py

Структура проекта

main.py — главный файл запуска, цикл игры

scenes/ — сцены и меню (меню, диалоги, мини-игры)

mechanics/ — механика страха, анимации, эффекты

dialogs/ — сценарий диалогов (например, в JSON)

assets/ — графика, громкие эффекты, музыка

utils/ — вспомогательные модули, сохранение, загрузка

Ключевые особенности и механики

Диалоги: с выбором из 5-7 ключевых решений во время каждой главы

Система страха: влияет на восприятие, меняет поведение мира (смешное, тревожное, пугающее)

Мини-игры: разные подходы к абсурдным действиям — готовка, реакция на шевеление в котлете

Концовки: 4+ (счастливая, смешная, мрачная, абсурдная, секретная)

Графика: 15+ локаций, 10+ персонажей с анимациями эмоций

Звук: фоновая музыка, спецэффекты (скрип, шёпот, бульканье)

План разработки и этапы

Пре-продакшн: 1.5 мес. (сценарий, концепты, музыка)

Разработка: 4 мес. (программирование, интеграция)

Пост-продакшн: 1 мес. (отладка, тестирование)
